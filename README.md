## 芯片介绍

STM32L151C8T6是一款基于ARM Cortex-M内核的32位微控制器，属于STM32系列。该芯片具有以下特性：

- **程序存储器容量**: 64KB
- **工作电压**: 2V ~ 3.6V
- **工作温度范围**: -40°C ~ 85°C
